Introduction¶

Le nombre de rendez-vous manqus des patients dans les centres de sant brsiliens est non ngligeable. Dans ce projet nous analyserons un ensemble de donnes de +100K rendez-vous mdicaux programms dans des structures de sant brsiliennes en vue de dcouvrir les tendances pouvant nous aider expliquer les raisons possibles pour lesquelles un patient quoique programm pour un rendez-vous mdical ne se prsente pas. Les questions qui suivent nous guiderons dans notre anlyse

  • Quelles sont les proportions respectives des patients qui se sont prsents ou non leur rendez-vous mdical ?

  • Les patients ne respecteraient-il pas leur rendez-vous par simple oubli ?

  • L'aide sociale serait-elle un facteur qui expliquerait le fait qu'un patient ne se prsente pas son rendez-vous ?

# Cette fonction nous permet d'afficher cte  cte le diagramme  barres (pour les effectifs) 
# et le diagramme circulaire pour le pourcentage de chaque effectif
def display_subplots(data_serie, plt1_params={}, plt2_params={}):
    plt.figure(figsize=(12,5))

    # Cration du premier sous graphique (diagramme  barre)
    plt.subplot(121)
    if plt1_params:
        # Dfinir le titre du graphique
        plt.title(plt1_params.get('title'))
        #Pour dfinir les libells des axes
        plt.xlabel(plt1_params.get('xlabel'))
        plt.ylabel(plt1_params.get('ylabel'))
        
    data_serie.plot(kind='bar', rot=0);

    # Cration du second sous-graphique (diagramme circulaire)
    plt.subplot(122)
    if plt2_params:
        # Dfinir le titre du graphique
        plt.title(plt2_params.get('title'))
        #Pour dfinir les libells des axes
        plt.xlabel(plt2_params.get('xlabel'))
        plt.ylabel(plt2_params.get('ylabel'))
    
    data_serie.plot(kind='pie', autopct='%.0f%%', label='', colors = ['tab:orange', 'tab:blue']);

    plt.legend();

# Cette fonction permet d'encoder des variable n-aire en associant  chaque valeur
#un mot/texte plus descriptif
def encode_bool_indexes(serie, encoding_dict={}):
    serie_encode = serie
    if(encoding_dict):
        serie_encode.rename(encoding_dict, inplace=True)
    return serie
